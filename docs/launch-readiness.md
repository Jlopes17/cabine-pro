# Launch Readiness Checklist

## Antes do beta fechado

- [ ] Protótipo abre no desktop.
- [ ] Protótipo abre no mobile.
- [ ] Fluxo de login demo funciona.
- [ ] Briefing completo é compreensível.
- [ ] Modo Cabine é legível.
- [ ] Modo Seguro é claro.
- [ ] Fontes e alertas são entendidos.
- [ ] Disclaimer editorial aparece.
- [ ] Feedback pode ser registrado no GitHub.

## Antes de cobrar

- [ ] CNPJ/forma de recebimento definida.
- [ ] Termos de uso revisados.
- [ ] Política de privacidade revisada.
- [ ] Política de cancelamento definida.
- [ ] Fonte/API com permissão escrita.
- [ ] Método de pagamento definido.
- [ ] Preço beta definido.
- [ ] Processo de suporte definido.

## Antes do SaaS público

- [ ] Auth real.
- [ ] Banco de dados.
- [ ] Auditoria persistente.
- [ ] Billing recorrente.
- [ ] Logs e monitoramento.
- [ ] Backups.
- [ ] Revisão legal/fiscal.
- [ ] Testes de segurança básicos.
- [ ] Teste com usuários reais concluído.
