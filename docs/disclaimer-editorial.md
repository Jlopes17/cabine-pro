# Disclaimer Editorial

Cabine Pro é uma ferramenta de apoio editorial para preparação de briefing esportivo.

O produto não substitui:

- fontes oficiais;
- checagem jornalística;
- responsabilidade editorial humana;
- decisão profissional de narradores, comentaristas e produtores;
- parecer jurídico;
- contrato/licença de dados esportivos.

## Uso correto

O usuário deve tratar o briefing como apoio de preparação, sempre observando o nível de confiança indicado.

Informações com selo **Verificar**, **Cuidado** ou **Conflito** não devem ser apresentadas ao vivo como fato confirmado.

## Uso proibido

- usar rumor como fato;
- remover fonte de informação sensível;
- copiar conteúdo protegido de terceiros;
- apresentar escalação provável como confirmada sem fonte oficial;
- apresentar lesão/suspensão sem confirmação;
- usar o produto para difamar, atacar ou espalhar desinformação.

## Mensagem curta para interface

> Cabine Pro ajuda a organizar e validar informações. A responsabilidade final pelo uso editorial é do usuário.
