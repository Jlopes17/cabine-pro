# Source Policy

## Política de fontes do Cabine Pro

O produto deve funcionar com uma política explícita de uso de fontes.

## Permitido

- Dados simulados no protótipo.
- Links manuais para fontes oficiais/editoriais.
- Resumo próprio de informações públicas.
- APIs contratadas/licenciadas dentro dos termos.
- Citação de fonte, título, horário e URL.

## Proibido sem autorização

- Scraping de Flashscore, SofaScore, ESPN, ge, UOL, Lance, Transfermarkt ou similares.
- Copiar texto de matérias.
- Republicar bases completas.
- Usar logos/escudos sem permissão.
- Vender dados brutos de terceiros.
- Apresentar rumor como fato confirmado.

## Requisitos para fontes comerciais

Antes de usar uma fonte/API em produto pago, registrar:

- link dos termos;
- preço;
- uso permitido;
- caching permitido;
- exportação PDF permitida;
- uso em SaaS subscription permitido;
- necessidade de atribuição;
- restrições de mídia/broadcast;
- data da revisão;
- evidência por escrito do fornecedor, se necessário.
