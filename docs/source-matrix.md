# Matriz de Fontes — Brasil

Esta matriz deve ser validada antes de qualquer lançamento pago.

## Permitido no protótipo fechado

| Fonte | Uso permitido agora | Observação |
|---|---|---|
| Dados simulados | Sim | Melhor para validar UX sem risco. |
| Links manuais | Sim | Guardar URL, título e horário. |
| Fontes oficiais | Sim, como referência | Não copiar bases completas. |
| Notícias editoriais | Sim, como referência | Resumo próprio, sem copiar texto. |

## Precisa confirmar/licenciar antes de vender

| Fonte/API | Status | Próxima ação |
|---|---|---|
| SportMonks | Candidata principal | Confirmar uso em SaaS editorial/subscription. |
| API-Football | Barata, mas exige cuidado | Pedir permissão por escrito para uso comercial. |
| football-data.org | Possível MVP simples | Confirmar cobertura Brasil e termos. |
| TheSportsDB | Auxiliar | Confirmar limites e atribuição. |
| Opta/Stats Perform | Futuro premium | Contato comercial quando houver receita. |
| Sportradar | Futuro premium | Contato comercial quando houver receita. |

## Bloqueado sem contrato ou autorização

| Fonte | Risco |
|---|---|
| Flashscore scraping | Alto risco de violar termos. |
| SofaScore scraping | Alto risco de violar termos. |
| ESPN/ge/UOL/Lance scraping | Risco de direito autoral e termos. |
| Transfermarkt scraping | Risco de base de dados protegida. |
| Logos/escudos oficiais | Risco de marca/direitos. |
| Matérias copiadas | Direito autoral. |

## Regra operacional

Cabine Pro deve vender **organização, confiança e rastreabilidade**, não dados copiados.
