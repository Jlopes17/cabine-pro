# Acceptance Criteria

## Produto geral

- O usuário entende o valor em menos de 10 segundos.
- O usuário consegue entrar na demo sem ajuda.
- O usuário consegue abrir um briefing completo.
- O usuário entende o score de confiança.
- O usuário entende o que não deve falar ao vivo.
- O usuário consegue abrir o Modo Cabine.

## Confiança

- Todo bloco crítico tem fonte ou aviso.
- Rumores não aparecem no Modo Seguro.
- Escalação provável não aparece como confirmada.
- Lesão não confirmada aparece como Verificar.
- Conflitos são sinalizados.

## Validação com usuário

- O usuário consegue explicar o produto com suas palavras.
- O usuário diz se usaria antes de uma transmissão.
- O usuário indica preço aceitável.
- O usuário aponta pelo menos uma melhoria concreta.

## Comercial

- Não vender publicamente sem termos, privacidade, fonte/licença e pagamento claros.
