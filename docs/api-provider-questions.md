# Perguntas para fornecedores de dados esportivos

Enviar antes de usar qualquer API em produto pago.

## Contexto para enviar

Estamos criando uma plataforma SaaS de briefing editorial pré-jogo para narradores, comentaristas, produtores e entusiastas. Não queremos revender dados brutos. Queremos usar dados estruturados para gerar briefings com fonte, confiança, auditoria e exportação em PDF/modo cabine. Usuários pagarão assinatura mensal.

## Perguntas obrigatórias

1. O uso dos dados em um SaaS de briefing editorial pago é permitido no plano escolhido?
2. Podemos exibir dados dentro de um briefing textual?
3. Podemos exportar o briefing em PDF para o usuário?
4. Podemos armazenar/cachear dados? Por quanto tempo?
5. Podemos exibir nomes de clubes, competições e jogadores?
6. Escudos/logos estão incluídos na licença?
7. Lesões, suspensões, lineups e standings têm restrições especiais?
8. Existe restrição para uso por profissionais de mídia, transmissão, rádio, TV, streaming ou criadores?
9. Existe limite por usuário final ou por produto?
10. Precisamos mostrar atribuição? Como?
11. Podemos usar os dados para gerar texto com IA?
12. Podemos usar a API em português do Brasil?
13. Quais competições do Brasil são cobertas?
14. Copa do Brasil, Brasileirão Série A/B, estaduais, Libertadores e Sul-Americana estão incluídas?
15. Existe custo extra para news, odds, xG, transfers, injuries ou lineups?

Salvar a resposta do fornecedor no GitHub antes de lançar o beta pago.
