# Testing Matrix

## Devices

- Desktop Chrome
- Desktop Safari
- Mobile Safari
- Mobile Chrome

## Main flows

- Landing to login
- Login to app
- Create briefing
- Open briefing
- Open Modo Cabine
- Filter sources
- Open policy modal
- View plans

## Critical trust tests

- Confirmed information is visually different from Verificar.
- Modo Seguro does not show fragile information.
- Disclaimer is accessible.
- Source policy is visible in docs.
- User can understand confidence score.
