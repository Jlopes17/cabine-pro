# Subscription Model — Cabine Pro

## Princípio

A Cabine Pro deve cobrar pelo valor de preparação, confiança editorial e economia de tempo, não por “texto gerado por IA”.

## Planos iniciais sugeridos para validação

### Briefing avulso

Preço inicial sugerido: **R$9–19 por jogo**.

Objetivo: validar demanda sem compromisso mensal.

### Entusiasta

Preço inicial sugerido: **R$19–29/mês**.

Para:

- fãs;
- pequenos criadores;
- podcasts;
- páginas de clubes;
- estudantes.

Limites:

- poucos briefings por mês;
- sem recursos avançados de equipe;
- fonte e confiança básicas.

### Pro fundador

Preço inicial sugerido: **R$49–79/mês** durante beta.

Depois de validação, possível ajuste para **R$99–149/mês**.

Para:

- narradores;
- comentaristas;
- produtores;
- criadores recorrentes.

Recursos:

- Modo Cabine;
- Modo Seguro;
- exportação PDF;
- auditoria;
- histórico;
- fontes e alertas.

### Equipe beta

Preço inicial sugerido: **R$199–399/mês**.

Para:

- rádios;
- canais;
- streamers;
- produtoras;
- equipes de mídia.

Recursos:

- múltiplos usuários;
- permissões;
- padrões editoriais;
- colaboração;
- suporte prioritário.

## Custos que influenciam preço real

- API esportiva licenciada;
- IA;
- hosting;
- banco de dados;
- pagamento;
- impostos;
- suporte;
- jurídico;
- licenças de dados;
- volume de briefings.

## Estratégia recomendada

1. Começar com protótipo e dados simulados.
2. Fazer beta manual ou semi-manual.
3. Cobrar founding plan barato.
4. Só automatizar mais depois de sinal de pagamento.
5. Ajustar preço com base em uso real e custo de fontes.
