# Cost Plan — Cabine Pro

## Fase 1 — Validação sem vender

Objetivo: provar que o produto é útil.

Custo estimado: **R$0–100 total**.

Itens:

- GitHub: R$0
- Deploy estático: R$0
- Protótipo: R$0
- Dados simulados: R$0
- Domínio opcional: R$40–80/ano

Não usar ainda:

- API esportiva paga;
- backend complexo;
- pagamento recorrente;
- jurídico pesado;
- scraping.

## Fase 2 — Beta manual/semi-manual

Objetivo: validar com usuários reais e talvez cobrar via Pix/manual.

Custo estimado: **R$100–500/mês**.

Itens possíveis:

- domínio;
- OpenAI para geração assistida;
- deploy;
- email simples;
- planilha/Notion/GitHub para operações.

## Fase 3 — MVP pago simples

Objetivo: login, histórico, billing e automação parcial.

Custo estimado: **R$500–2.000/mês**.

Itens:

- frontend;
- backend/database;
- IA;
- email;
- pagamentos;
- uma API esportiva barata ou plano inicial;
- contador/CNPJ se vender oficialmente.

## Fase 4 — SaaS profissional

Objetivo: produto estável, usuários pagantes, suporte e dados licenciados.

Custo estimado: variável, possivelmente **R$3.000+/mês** dependendo de dados, IA e suporte.

## Regra financeira

Não assumir custo alto antes de provar:

1. usuário usa;
2. usuário confia;
3. usuário pagaria;
4. dados/licenças são viáveis;
5. CAC e suporte fazem sentido.
