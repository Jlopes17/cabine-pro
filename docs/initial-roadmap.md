# Initial Roadmap

## Semana 1 — Validação visual e fluxo

- Publicar protótipo.
- Testar desktop e mobile.
- Ajustar bugs de navegação.
- Melhorar clareza da landing.
- Revisar Modo Seguro.

## Semana 2 — Validação com usuários

- Enviar para 3–5 narradores/produtores/criadores.
- Registrar feedback.
- Descobrir preço aceitável.
- Ver quais blocos são indispensáveis.

## Semana 3 — Beta manual

- Escolher 3 jogos reais.
- Montar briefings semi-manuais.
- Entregar PDF/modo cabine.
- Medir tempo economizado.

## Semana 4 — Decisão MVP

- Definir se vale construir backend.
- Escolher stack.
- Escolher fonte/API candidata.
- Definir preço beta.
- Criar plano de assinatura fundador.
