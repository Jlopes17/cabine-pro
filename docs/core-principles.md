# Core Principles

1. Segurança editorial antes de velocidade.
2. Fonte antes de texto bonito.
3. Confiança explícita, não implícita.
4. Informação frágil nunca vira fato.
5. Modo Seguro deve proteger o usuário ao vivo.
6. O produto começa simples e semi-manual.
7. Automatizar só depois de validar valor.
8. Guardar evidência de permissões de dados.
9. Cobrar só quando o fluxo entregar valor real.
